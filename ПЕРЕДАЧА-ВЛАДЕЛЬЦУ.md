## 2. Что нужно сделать вам (новому владельцу) — по шагам

### Шаг A. Свои аккаунты (обязательно)

1. Зарегистрируйтесь на **GitHub**: https://github.com/signup  
2. Зарегистрируйтесь на **Vercel** (можно через GitHub): https://vercel.com/signup  

Без своих аккаунтов вы не сможете полноценно владеть хостингом.

### Шаг B. Принять репозиторий с кодом (GitHub)

Текущий владелец делает **один** из вариантов:

**Вариант 1 — полная передача (рекомендуется)**  
GitHub → репозиторий `dc` → **Settings** → внизу **Danger Zone** → **Transfer ownership** → указать ваш логин GitHub.

**Вариант 2 — совладение**  
**Settings** → **Collaborators** → **Add people** → ваш логин → роль **Admin**.  
(Код ваш, но репозиторий формально остаётся у передающего, пока не сделает Transfer.)

После передачи вы клонируете:

```bash
git clone https://github.com/ВАШ-ЛОГИН/dc.git
cd dc
```

### Шаг C. Принять хостинг (Vercel)

Текущий владелец:

1. Заходит на https://vercel.com → проект **debit-cards-ru**  
2. **Settings** → **General** → внизу **Transfer Project**  
3. Указывает ваш аккаунт Vercel (email или команду).

Вы принимаете приглашение в почте / в Vercel Dashboard.

**Либо** вы сами подключаете сайт заново (если Transfer недоступен):

1. Vercel → **Add New** → **Project**  
2. Import из **вашего** GitHub-репозитория `dc`  
3. Framework: **Other**, Root: `/`, Build: пусто, Output: `/`  
4. Deploy → получите свой URL вида `https://ваш-проект.vercel.app`  
5. В **Settings → Domains** можно привязать свой домен.

### Шаг D. Свои партнёрские ссылки (важно для денег)

Сейчас кнопки «Оформить» ведут в **getlink.su** с чужим `aff_id=221509`.  
Пока вы не замените ссылки, **конверсии и выплаты идут прежнему вебмастеру**.

Откройте файл **`offers.js`**, в каждом оффере поле `url` замените на **свои** ссылки из партнёрки (getlink / finhelper / другая сеть).

Пример полей в ссылке getlink:

- `aff_id` — ваш ID вебмастера  
- `pltfm_id` — ID площадки  
- `offer_id` — ID оффера в сети  

Офферы на витрине сейчас:

| Банк | offer_id (getlink) |
|------|-------------------|
| МТС Банк «Скидка везде» | 10842 |
| ВТБ МИР | 10356 |
| ПСБ «Твой Кешбэк» | 9767 |
| МТС Деньги | 11443 |

После правки — commit + push в GitHub; Vercel обновит сайт за 1–2 минуты (если репозиторий подключён).

### Шаг E. Отключить старые копии (по желанию передающего)

После вашего Vercel-деплоя передающий может:

- удалить проект `debit-cards-ru` у себя в Vercel;  
- отключить GitHub Pages у репозитория `dc` или передать репозиторий вам полностью.
